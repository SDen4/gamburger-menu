##  Simple gamburger menu:
```sh
$ git clone https://github.com/SDen4/gamburger-menu.git
$ cd gamburger-menu
$ npm i
$ git init
$ git remote set-url origin <link of new repo in github>
$ git push -u origin master
$ npm run dev
```
